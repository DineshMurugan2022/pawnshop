# Jewelry Store Website

Welcome to the Jewelry Store Website project! This application is designed to provide users with an elegant platform to browse and purchase jewelry items.

## Project Structure

The project is organized as follows:

```
jewelry-store-website
├── src
│   ├── components
│   │   ├── Navbar.tsx        # Navigation bar component
│   │   └── BrowseButton.tsx  # Button to navigate to the Browse page
│   ├── pages
│   │   ├── Home.tsx          # Landing page component
│   │   ├── Login.tsx         # User login form component
│   │   └── Browse.tsx        # Component to display available jewelry items
│   ├── App.tsx               # Main application component with routing
│   └── index.tsx             # Entry point of the application
├── public
│   └── index.html            # Main HTML file for the application
├── package.json              # npm configuration file
├── tsconfig.json             # TypeScript configuration file
└── README.md                 # Project documentation
```

## Features

- **Home Page**: A welcoming landing page for users.
- **Login Page**: A secure login form for user authentication.
- **Browse Page**: A dedicated page for browsing jewelry items.
- **Navigation Bar**: A consistent navigation experience across the application.
- **Browse Button**: A button labeled "Browse Jewel" that directs users to the Browse page.

## Getting Started

To get started with the project, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd jewelry-store-website
   ```

3. Install the dependencies:
   ```
   npm install
   ```

4. Start the development server:
   ```
   npm start
   ```
